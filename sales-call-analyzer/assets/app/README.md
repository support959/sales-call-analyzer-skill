# מנתח שיחות מכירה — הכלי המקומי

כלי קטן שרץ על המחשב שלך. זורקים לתוכו הקלטת שיחת מכירה / אבחון (או מדביקים תמליל) ומקבלים דוח
ניתוח מלא בעברית: סיכום ותובנות, ציון איכות לכל חלק בשיחה, התנגדויות שעלו, ומשימות המשך.

## דרישות

- **Node.js 18 ומעלה** — מ-https://nodejs.org (כפתור LTS). בדיקה: `node --version`.
- **מפתח Anthropic** — חובה, מ-https://console.anthropic.com ← API Keys.
- **מפתח OpenAI** — רק אם רוצים לנתח הקלטות אודיו, מ-https://platform.openai.com/api-keys.

## הפעלה ראשונה (פעם אחת)

1. העתיקו את הקובץ `.env.example` לשם `.env`.
2. פתחו את `.env` והדביקו את המפתחות שלכם.
3. הריצו מתוך התיקייה הזו:

   ```
   node server.mjs
   ```

4. ייפתח כתובת מקומית (ברירת מחדל `http://localhost:4848`). פתחו אותה בדפדפן.

## שימוש יומיומי

1. `node server.mjs`
2. פותחים את הכתובת בדפדפן.
3. גוררים קובץ הקלטה, או מדביקים תמליל, ולוחצים **"נתח את השיחה"**.
4. מקבלים דוח — אפשר להדפיס/לשמור כ-PDF או להעתיק.

לבדיקה ראשונה בלי הקלטה: בלשונית "טקסט" לחצו **"טען שיחת דוגמה"** ואז "נתח את השיחה".

## פרטיות

- הכל רץ במחשב שלך. ההקלטה/התמליל נשלחים **רק** לשירות התמלול (OpenAI) ולשירות הניתוח (Anthropic),
  עם המפתחות שלך — ולא לשום מקום אחר.
- קובץ `.env` (המפתחות) וקבצי הקלטה **לא** נשמרים בגיט (מוגדר ב-`.gitignore`).

## מגבלות ידועות

- תמלול אודיו מוגבל ל-25MB לקובץ. שיחה ארוכה? ייצאו כ-MP3 מונו ~64kbps והקובץ יהיה קטן בהרבה.
- איכות התמלול תלויה באיכות ההקלטה. רעש רקע פוגע בתוצאה.
